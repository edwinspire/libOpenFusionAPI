libOpenFusionAPI Dynamic API Runtime Platform
Fastify-powered API Engine

Ship configurable APIs with production-ready control.

libOpenFusionAPI is a modular runtime that exposes HTTP and WebSocket endpoints from dynamic configuration. It supports multiple execution handlers including SQL, JS, SOAP, MCP, and MongoDB, making it ideal for integration-heavy systems.

Open Documentation Server Health

Core Capabilities

  • Endpoint lookup and execution at runtime with cache acceleration.
  • Pluggable handlers for database, script, webhook, and MCP workflows.
  • Authentication and policy hooks per endpoint definition.
  • Built-in tracing, request logging, and response lifecycle events.

Technology Base

Built on Node.js, Fastify, and Sequelize with multi-environment operation for dev, qa, and production. Designed for teams that need speed without losing governance or observability.
